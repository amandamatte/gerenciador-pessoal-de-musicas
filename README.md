🎵 Gerenciador de Músicas e Playlists 🎵

Projeto desenvolvido em Java para a disciplina de Fundamentos de Programação. O sistema permite cadastrar artistas, álbuns, músicas e playlists, com funcionalidades de inserção e listagem.

Para executar, compile todos os arquivos com `javac *.java` e rode com `java Main`. O menu principal guia todas as opções disponíveis.

Aproveite! 🎵
